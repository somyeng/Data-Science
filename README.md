### 2. 전국 범죄 발생 및 검거 현황 분석
- 소스 코드 : [바로가기](https://github.com/somyeng/Data-Science/blob/main/source_code/02.%20%EC%A7%80%EC%97%AD%EB%B3%84%20%EB%B2%94%EC%A3%84%20%ED%98%84%ED%99%A9%20%EB%B6%84%EC%84%9D.ipynb)
